# Event Ticket Booking System

## 📌 Project Overview

The **Event Ticket Booking System** is a simple web-based application that allows users to book tickets for an event. Users can select the number of tickets, view the ticket price, and see the total amount automatically calculated.

## 🎯 Objectives

* Provide a simple interface for booking event tickets
* Display ticket price information
* Calculate the total price based on selected tickets

## ✨ Features

* User-friendly interface
* Ticket price display
* Automatic total price calculation
* Simple and interactive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📁 Project Structure

EventTicketBookingSystem/
│
├── index.html

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in a web browser.
4. Select the number of tickets and view the total price.
